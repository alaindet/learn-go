# Math

The `math` package includes methods to perform mathematical calculations, most common functions are

- `math.Abs(val)`
- `math.Ceil(val)`
- `math.Copysign(x, y)`
- `math.Floor(val)`
- `math.Max(x, y)`
- `math.Min(x, y)`
- `math.Mod(x, y)`
- `math.Pow(x, y)`
- `math.Round(val)`
- `math.RoundToEven(val)`

Some constants
- `math.MaxInt8`
- `math.MinInt8`
- `math.MaxInt16`
- `math.MinInt16`
- `math.MaxInt32`
- `math.MinInt32`
- `math.MaxInt64`
- `math.MinInt64`
- `math.MaxUint8`
- `math.MaxUint16`
- `math.MaxUint32`
- `math.MaxUint64`
- `math.MaxFloat32`
- `math.MaxFloat64`
- `math.SmallestNonZeroFloat32`
- `math.SmallestNonZeroFloat64`
